# ts-kicad-repository

This repository contains the package index for Tecsmith KiCAD libraries and plugins.

Currently included:

* [ts-kicad-lib](https://github.com/Tecsmith/ts-kicad-lib)
* [ts-kicad-plugins](https://github.com/Tecsmith/ts-kicad-plugins)

## To use and install in KiCAD

Start up the **Plugin and Content Manager**.

Click on the "Manage..." button and then "+" *(Add)*.  Use the following:
| | |
| --- | --- |
| Name | Tecsmith repository |
| URL | `https://github.com/Tecsmith/ts-kicad-repository/raw/main/repository.json` |
